# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

- Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Our Standards

- Be respectful and constructive.
- Assume good intent and focus on technical outcomes.
- Avoid harassment, discrimination, and personal attacks.

## Enforcement

To report unacceptable behavior, contact the maintainers via repository issues:

- https://github.com/ayutaz/uStyle-Bert-VITS2/issues

Project maintainers are responsible for clarifying and enforcing this policy.
